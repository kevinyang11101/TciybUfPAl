## 项目概述

医院管理系统是一款基于 JavaWeb 技术的医疗信息管理系统。系统采用 B/S 结构，结合 SpringBoot 和 Vue 技术，以 MySQL 作为后台数据库，实现了医院日常业务流程的信息化、数字化管理。
